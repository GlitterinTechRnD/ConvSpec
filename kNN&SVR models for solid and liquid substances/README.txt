Please ensure that MATLAB 2019 or a later version is installed on your computer.

The data files Glucose_data.xlsx and Plastic_data.xlsx must not be modified in terms of data positions or formats; otherwise, the program may encounter errors.

The main program scripts are:

Main_Glucose_solution.m: Script for glucose solution modeling and testing
Main_Plastic.m: Script for plastic classification testing

To run a script, simply open it in the MATLAB Editor, and click Run to view the corresponding test results.
